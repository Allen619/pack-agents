## 已知问题和限制

### 当前限制

- 暂不支持定时执行工作流（计划 v1.1.0 支持）
- 批量操作功能有限（计划 v1.2.0 增强）
- 需要手动配置 API 密钥（计划 v1.1.0 提供向导）

### 兼容性说明

- 需要 Node.js 18.0.0 或更高版本
- 推荐使用 Chrome 90+、Firefox 88+、Safari 14+
- 需要稳定的网络连接访问 AI 服务
